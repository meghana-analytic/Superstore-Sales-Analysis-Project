📊 **Superstore Sales Analysis Project**
📌 **Project Overview**
This project is an Excel-based sales analysis of a Superstore dataset.
The goal is to analyze sales, profit, customer segments, and regional performance using pivot tables and dashboards.
🛠️ **Tools Used**
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Slicers
• Dashboard Design.
📂 **Project Files**
• Superstore dataset (Excel file)
• Cleaned data sheet
• Analysis sheet (Pivot tables & insights)
• Dashboard sheet.
📊 **Key Insights**
• Technology category generates the highest profit
• West region performs better in both sales and profit
• Consumer segment contributes the most sales
• Some high-sales products have low profit margins
• Central region shows weaker performance compared to others.
📈 **Dashboard Features**
• KPI cards (Total Sales, Profit, Orders, Profit %)
• Sales by Category
• Regional performance analysis
• Monthly sales trend
• Top-performing sub-categories
• Interactive slicers (Region, Category, Segment).
🎯 **Skills Demonstrated**
• Data cleaning and preparation
• Data visualization in Excel
• Business insights generation
• Dashboard creation
• Analytical thinking.
🚀 **Conclusion**
This project helps understand how sales and profit vary across regions, categories, and customer segments, and provides insights for better business decisions.
